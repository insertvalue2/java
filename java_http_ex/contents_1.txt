


JSON 이란 

JSON(javaScript Object Notation)의 약자로 JavaScript에서 객체를 만들때 
사용하는 표현식을 말한다.

var person = {"height" : 180, "job" : "programmer"}
var members = ["홍길동", "김유신" , "티모"];

클라이언트 - 서버 프로그래밍을 할 때 

자바 스크립트로 만든 객체를 php나 jsp 또는 자바로 연동할 때  배열이나 객체상태 
그대로 전송할 수 있는 방법이 존재하지 않습니다. 
자바의 경우 문자열 형태로 받아서 직접 가공해서 사용해야 하는데 이 부분이 
매우 불편한 방식입니다. 
 
규칙, 약속, 표준,을 정하고 그 이름을  JSON이라고 부릅니다.   

why 
이 표현식은 사람도 이해하기 쉽고 기계도 이해하기 쉬우면서 데이터의 용량이 적다
이런 이유로 프로그램의 설정의 저장이나 데이터를 전송하는 일에 많이 사용된다.
